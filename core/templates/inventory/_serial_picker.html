{% comment %} Lightweight serial picker. For serial-tracked products it turns the free-text serial input into a datalist-backed selector populated from /inventory/serials/ options/ (tenant + permission scoped); for non-serial products it disables the field. Backend cardinality validation (package 1) remains the source of truth — this only makes selection easy and hides unavailable serials. Works with both plain forms (name="product"/"serial_number") and formset rows (…-product / …-serial_number), and with rows added dynamically. {% endcomment %}
